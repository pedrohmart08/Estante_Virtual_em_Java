# EstantX Virtual
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)
Aplicação de desktop para organização de livros, você pode cadastrar, remover, atualizar e buscar seus livros.

## Funcionalidades
-  Cadastro de livros com título, autor, ISBN, gênero
-  Busca por título ou autor
-  Edição completa dos dados
-  Remoção com confirmação

## Tecnologias
- Java 17
- JavaFX 21.0.11
- Apache Maven 3.8.7
- CSS

## Instalação e execução

Com o Java (verifique se é a versão 17+) já instalado em sua máquina. Abra o terminal dentro da pasta do projeto e digite:

```bash
#Verifique a versão do java
java -version
#Verifique a versão do Maven e se está instalado
mvn -version
#Limpa e compila o projeto
mvn clean install
#Roda a aplicação
mvn javafx:run
```
